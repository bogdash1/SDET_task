О проекте:

Файл: conftest.py

  В данном файле происходит создание фикстуры (fixture) на уровне сессии для запуска браузера.

Файл: BaseApp.py

  Файл представляет собой класс BasePage с набором методов для взаимодействия с веб-страницами с использованием библиотеки Selenium. 
Функция def find_elements(self, locator,time=10) определена в программе, но в рамках требуемых задач неиспользуется - необходима для возможности изменения фрагментов кода, без изменения базового класса.

Файл: MainPage.py

  В данном файле происходит объявление необходимых локаторов и определяются функции, необходимые для парсинга тестового сайта.

Файл: tests.py

  В данном файле происходит запуск кода для дальнейшего тестирования web-страницы согласно заданию. Вводные данные определяются на уровне кода, передаются в явном виде. В результате выполнения кода мы получаем отчет, корректно ли отработала web-страница или произошла ошибка (генерируется исключение). Реализованы механизмы проверки адреса электронной почты и номера телефона на валидность согласно заданию. В случае ввода невалидных данных генерируется исключение.
